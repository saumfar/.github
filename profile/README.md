# Saumfar

Compliance tooling for Norwegian architects. Based in Bergen.

We structure planbestemmelser from all 357 kommuner into machine-readable rules and check them against TEK17, SAK10, and the relevant reguleringsplan for a given property. The goal is to save architects time between sketch phase and byggesøknad.

69,500+ bestemmelser extracted so far. Still pre-launch.

Laravel, React, TypeScript, MapLibre GL JS, PostGIS, Cloudflare Workers.
