# Saumfar

Compliance tooling for Norwegian architects. Based in Bergen.

We structure regulations from all 357 municipalities into machine-readable rules and check them against TEK17, SAK10, and the relevant regulations plans for a given property. The goal is to save architects time between sketch phase and building application.

69,500+ regulations extracted so far. Still pre-launch.

Laravel | React | TypeScript | MapLibre GL JS | PostGIS | Cloudflare Workers
